# PicSafe 📸🔒

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/Platform-iOS%20%7C%20Android-blue)](https://github.com/rohitpidishetty/PicSafe)

**PicSafe** is a secure and user-friendly mobile application for storing, managing, and sharing personal photos. It ensures your images remain private with end-to-end encryption and offline access.

---

## 🚀 Features

- 🔒 **Secure Storage**: All photos are encrypted before saving.
- 📁 **Custom Albums**: Organize your images in albums.
- ⚡ **Offline Access**: Manage your gallery without internet.
- 👤 **User Authentication**: Safe login system.
- ✨ **Intuitive UI**: Clean, Apple-inspired design.
- 📱 **Cross-Platform**: Works on both iOS and Android.

---

## 🛠 Tech Stack

- **Frontend**: React, Capacitor
- **Backend**: Firebase Realtime Database & Cloud Storage
- **Security**: AES/RSA End-to-End Encryption
- **Platform Support**: Android & iOS

---

## 💻 Installation

### Prerequisites

- Node.js >= 16.x  
- Ionic CLI  
- Android Studio / Xcode (for mobile builds)

### Steps

```bash
# Clone repository
git clone https://github.com/rohitpidishetty/PicSafe.git
cd PicSafe

# Install dependencies
npm install

# Run in development mode
npm start
npm run build
